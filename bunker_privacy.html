2.2 Datos técnicos y de uso (a través de Google: AdMob y Firebase)2.2 Technical and usage data (via Google: AdMob and Firebase)

Para anuncios, medición y estabilidad, la App utiliza servicios de Google que pueden procesar información técnica y de uso. Ejemplos típicos de datos tratados por estos servicios:

For ads, measurement, and stability, the App uses Google services that may process technical and usage information. Typical examples of data handled by these services include:

  • Identificadores: identificador publicitario de Android (Advertising ID) y/o identificadores de instancia del SDK.
  • Datos del dispositivo: modelo, versión de Android, idioma, zona horaria aproximada, datos de rendimiento.

  • Datos de red: IP y datos técnicos necesarios para entregar anuncios y servicios (procesamiento por el proveedor).
  • Identifiers: Android Advertising ID and/or SDK instance identifiers.
  • Device data: model, Android version, language, approximate time zone, performance data.
  • Usage data: general events (e.g., aggregated screens/actions), interaction metrics, and sessions.
  • Diagnostics: crash reports, stack traces, and performance signals to fix errors.
  • Network data: IP and technical data needed to deliver ads and services (processed by the provider).

5) Notificaciones (Firebase Cloud Messaging)5) Notifications (Firebase Cloud Messaging)

FCM

La App puede enviar notificaciones mediante Firebase Cloud Messaging (FCM), por ejemplo, comunicados generales. El mecanismo de entrega lo provee Google/Firebase.

The App may send notifications via Firebase Cloud Messaging (FCM), for example, general announcements. Delivery is provided through Google/Firebase infrastructure.

5.1 Topics y tokens5.1 Topics and tokens

Para entregar notificaciones, FCM utiliza un token de mensajería asociado a la instalación de la App en tu dispositivo. Este token se gestiona como parte del servicio de FCM.

To deliver notifications, FCM uses a messaging token associated with the App installation on your device. This token is managed as part of the FCM service.

  • El Desarrollador no opera un servidor propio para almacenar tokens ni administrar una base de datos de dispositivos.
  • La suscripción a un topic se gestiona a través del servicio de FCM (infraestructura de Google).
  • The Developer does not operate its own server to store tokens or maintain a device database.
  • Subscription to a topic is handled through the FCM service (Google infrastructure).

5.2 Control de notificaciones5.2 Notification controls

Podés desactivar notificaciones desde la configuración de Android para la App.

You can disable notifications for the App in Android settings.
